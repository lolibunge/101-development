¿Qué es la Programación Web?

Para mí la programación es escribir código para crear sitios web y aplicaciones que usamos todos los días.

Como por ejemplo: YouTube, Gmail, Shopify — todas usan programación web.

¿Qué compone un sitio web?

HTML = Estructura (como los simientos de una casa)
CSS = Estilo (colores, diseño)
JavaScript = Comportamiento (la electrica de al casa)

¿Cómo funcionan los sitios web?

Un sitio web vive en un servidor.
Frontend
Lo ves con un navegador.
Server
Base de datos
El frontend es lo que ves, y el backend es la lógica detrás.

¿Qué hacen los desarrolladores web?

Desarrollador Frontend – diseña apariencia e interacciones
Desarrollador Backend – maneja datos y lógica
Desarrollador Full-Stack – hace ambas cosas

¿Tú también puedes aprenderlo?

¡Sí! No necesitas ser experta/o en matemáticas. Solo necesitas curiosidad, acceso a internet y paciencia.
“No tienes que ser genial para comenzar, pero debes comenzar para llegar a serlo.”

Estilizando con CSS 🎨

CSS le da estilo a tu sitio: colores, tamaños, fuentes, márgenes... ¡y mucho más!

Por ejemplo:

      <style>
        h1 {
          color: blue;
          font-family: sans-serif;
        }
      </style>
